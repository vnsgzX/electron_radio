Electron Radio
A simple desktop radio application built with Electron.

🚀 Description
This is a lightweight and fast desktop application for streaming your favorite radio stations. The project was created with the goal of being easy to use and efficient.

🛠 Technologies
Electron

JavaScript

HTML5 & CSS3

📦 How to run the project
Before you begin, make sure you have Node.js installed.

Clone the repository:
git clone https://github.com/vnsgzX/electron_radio.git

Navigate to the project folder:
cd electron_radio

Install the dependencies:
npm install

Start the application:
npm start
